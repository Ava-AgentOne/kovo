# Kovo 👾

Your personal AI agent. Self-hosted on Ubuntu, talks via Telegram.

## Install

```bash
wget -O bootstrap.sh https://raw.githubusercontent.com/Ava-AgentOne/kovo/main/bootstrap.sh
bash bootstrap.sh
```

Bootstrap clones the source code from GitHub, installs all dependencies, and sets everything up. Then fill in your tokens and start the service.
